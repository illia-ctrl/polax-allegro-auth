Allegro stores — 3-way comparison matched by SKU · private

Product = one unique SKU (the match key). The Where column shows each store's listing status: ✓ active (live, buyable) · ◌ inactive (listing exists but a draft, not live) · ⊘ ended · ✗ none (no listing). Hover any badge for details. Total products counts every SKU with any listing; “active in …” counts only buyable ones.

Each store cycles – any → ✓ active → ◌ inactive → ⊘ ended → ✗ none. Examples: Mlot ✗ = no listing on Mlot · Mlot ◌ = draft sitting on Mlot · Polax ✓ + Sila ✓ + Mlot ✗ = active on both, missing from Mlot. Issue chips (DUP/EAN/NAME/PID) narrow further. Reset clears all.
